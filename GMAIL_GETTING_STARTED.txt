KEY FEATURES

✅ Pattern Detection
   Type: "analisa cv dari inbox email xxx@gmail.com"
   System automatically understands and responds

✅ OAuth Authentication
   Secure Google login, read-only Gmail access
   No password storage, industry standard flow

✅ Smart Search
   Searches email subjects and file attachments
   Supports PDF, DOC, DOCX files

✅ Intelligent Analysis
   Uses GPT-4-turbo for CV analysis
   Extracts skills, experience, education

✅ Relevance Scoring
   Calculates match score (0-100%)
   Based on keyword relevance to CV content

✅ Beautiful Results
   Shows top 3 candidates
   Skills as colored badges
   Professional card layout

TROUBLESHOOTING

Problem: OAuth credentials missing
Solution: Check .env.local has all 3 variables

Problem: Cannot authenticate
Solution: Verify redirect URI in Google Console matches exactly

Problem: No matching emails found
Solution: Try different keyword or check inbox has emails with attachments

Problem: PDF extraction fails
Solution: PDF might be image-only, try another file

Problem: Slow processing
Solution: Normal, GPT-4 analysis takes 3+ seconds
